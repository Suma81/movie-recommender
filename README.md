# AI Movie Recommendation System 🎬

This project is a machine learning based movie recommender built using Python and Flask.

## Features
- Content-based movie recommendation
- Smart search
- Movie posters using TMDB API
- Interactive web interface

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- TMDB API

## How It Works
The system analyzes movie metadata such as genres, cast, keywords, and overview to calculate similarity between movies using cosine similarity.

Example:
If a user selects a movie like Inception, the system recommends similar movies.
